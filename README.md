# 🃏 CSS Playing Cards - 12 Card Set

A front-end project focused on recreating a deck of playing cards using modern CSS techniques. This project demonstrates layout precision, positioning, and responsive design.

## 🚀 Features

- **12 Cards:** Ace through 10, plus Jack and Queen.
- **Flexbox Layout:** Used to center the cards and make them wrap perfectly on any screen size.
- **CSS Grid:** Used to mathematically align the "pips" (symbols) in the middle of each card.
- **Absolute Positioning:** Used for the rank and suit in the card corners.

## 🛠️ Built With

- **HTML5:** Semantic structure.
- **CSS3:** Flexbox, Grid, and Absolute Positioning.

## 🧠 What I Learned

During this project, I mastered the difference between:

1. **Relative vs. Absolute Positioning:** Pinned the corner ranks so they stay inside the card.
2. **Flexbox vs. Grid:** Used Flexbox for the "big picture" (the table) and Grid for the "details" (the card faces).
3. **CSS Transforms:** Rotating the bottom-right corner 180° for an authentic card look.
